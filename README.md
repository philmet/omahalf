# Omahalf

https://philmet.github.io/omahalf

Website for Omahalf — an unofficial half marathon run by some friends starting at Omaha Beach, New Zealand, and finishing in Matakana Village.

## What is this

A single-page static website (`index.html`) with no dependencies, no build step, and no framework. Just open it in a browser.

## The run

- **Start:** Omaha Beach, Northland, NZ
- **Finish:** Matakana Village
- **Distance:** 21.14 km
- **Route:** Coast north to Point Wells, then back via the Matakana Cycle Trail
- **Elevation:** 101 m gain — flat for 15 km, one annoying hill at 17 km, then done

## Running locally

```bash
open index.html
```

That's it.

## Deploying

It's a single HTML file. Chuck it anywhere — GitHub Pages, Netlify, Vercel, a USB stick.

**GitHub Pages:**
1. Push to a repo
2. Go to Settings → Pages → set source to `main` / `root`
3. Done
